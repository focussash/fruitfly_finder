CUSTOM LEVEL IMAGES
===================

Add your own images to this folder to use them in the game!

HOW TO ADD IMAGES
-----------------

1. Enable "Use Local Images" in the game Settings

2. Add images to this folder using one of these naming patterns:

   NUMBERED (any theme):
   - level-1.jpg
   - level-2.png
   - level-3.webp

   THEMED:
   - kitchen-1.jpg, kitchen-2.png, etc.
   - garden-1.jpg, garden-2.png, etc.
   - fantasy-1.jpg, fantasy-2.png, etc.
   - retro-1.jpg, retro-2.png, etc.

3. Restart the game (or toggle the setting off/on)

SUPPORTED FORMATS
-----------------
- .jpg / .jpeg
- .png
- .webp

TIPS
----
- Images work best at 800x600 or similar aspect ratio
- Busy, detailed images make the game more challenging
- The game will use your images in order (level-1 for level 1, etc.)
- If you have fewer images than levels, they will cycle/repeat
